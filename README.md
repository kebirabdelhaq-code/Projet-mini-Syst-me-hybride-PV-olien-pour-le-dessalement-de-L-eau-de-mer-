# Projet-mini-Syst-me-hybride-PV-olien-pour-le-dessalement-de-L-eau-de-mer-Description du projet
Ce projet consiste à développer une application Python dotée d’une interface graphique (Tkinter) permettant de dimensionner un système hybride combinant l’énergie solaire photovoltaïque et l’énergie éolienne pour alimenter une unité de dessalement de l’eau de mer.

L’outil permet de :

calculer l’énergie quotidienne requise selon le procédé de dessalement (RO, MED, MSF) ;

estimer le nombre de panneaux photovoltaïques et d’éoliennes nécessaires ;

générer automatiquement plusieurs scénarios de répartition énergétique (100% PV, 100% éolien, 50/50, 70/30, 30/70, etc.) ;

comparer la performance et les coûts de chaque scénario ;

identifier le scénario optimal en termes de coût d’investissement ;

visualiser les courbes horaires de production solaire et éolienne, directement intégrées dans l’application.

Grâce à cet outil, il devient possible d’effectuer une étude préliminaire technico-économique rapide et intuitive d’un système hybride dédié à la production d’énergie pour des unités de dessalement, particulièrement utile dans les régions côtières ou isolées où les énergies renouvelables représentent une alternative efficace et durable.
